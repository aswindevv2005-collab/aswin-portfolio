# 🚀 Aswin Dev V - JARVIS AI Portfolio 🌌

Welcome to the future of professional portfolios! Built with **JARVIS**, a custom-trained AI assistant, this project is a fully interactive, conversational resume designed to showcase technical skills, projects, and personal journey in a futuristic HUD-style interface.

---

## ✨ Key Features
- **🤖 JARVIS AI Assistant**: A conversational AI interface that knows everything about Aswin's resume.
- **🖼️ Cinematic Galleries**: Interactive photo galleries showcasing childhood and present-day photography.
- **🛠️ IoT & Full-Stack Showcase**: Detailed project cards for IoT Pet Feeders, Movie Booking Apps, and Deep Learning models.
- **✨ Premium UI/UX**: Built with Glassmorphism, Neon HUD effects, and smooth Framer Motion animations.
- **📱 Responsive by Design**: Fully optimized for mobile HUD-style interactions.

- **Styling**: Tailwind CSS, Vanilla CSS (HUD Effects)
- **Animations**: Framer Motion, Lucide Icons
- **AI Core**: custom-prompted LLM integration via Next.js API Routes

## 🚀 Quick Start (Local Setup)

1. **Clone the repo:**
   ```bash
   git clone https://github.com/aswindevv2005-collab/aswin-portfolio.git
   cd aswin-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment:**
   Create a `.env.local` file and add your GEMINI_API_KEY (optional, falls back to simulated AI).

4. **Launch Dev Server:**
   ```bash
   npm run dev
   ```
